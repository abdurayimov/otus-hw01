# Домашнее задание №05

Hаписать свой тестовый фреймворк. Поддержать **аннотации**:
> @Test

> @Before

> @After

и **ассерты**:
> Assert.fail()

> Assert.true(...)

> Assert.notNull(...)

Запускать вызовом статического метода с (1) массивом классов с тестами, (2) именем package в котором надо найти и запустить тесты.
